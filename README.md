# Шахматный симулятор (ООП версия)

## Описание проекта
Консольная реализация шахмат на Python с использованием объектно-ориентированного подхода. 
Проект выполняется в рамках курса "Практикум по программированию".

## Структура проекта
- `chess_game.py` — основной код игры
- `plan.ipynb` — календарный план работ
- `README.md` — описание проекта

## Классы и архитектура
- `Piece` (абстрактный) + наследники: Pawn, Rook, Knight, Bishop, Queen, King
- `Board` — игровая доска
- `Move` — представление хода
- `Game` — логика игры

## Дополнительные возможности (будут реализованы)
- Откат ходов (undo)
- Подсказка доступных ходов
- Подсветка фигур под боем
- Взятие на проходе и превращение пешки
- 3 новые фигуры

## Инструкция по запуску
1. Склонировать репозиторий
2. Запустить `python chess_game.py`

## План работы
Подробный календарный план находится в файле `plan.ipynb`
